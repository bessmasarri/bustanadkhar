# 🌸 Bustan Al-Athkar (بستان الأذكار)

Welcome to my project! **Bustan Al-Athkar** is a kid-friendly web app designed to make learning and reading daily Adhkar (Islamic reminders) a fun and rewarding adventure. 

## What is this app?
I built this app to help children stay connected with their daily Dhikr in a way that feels like a game. Instead of just reading text, they are growing a **virtual garden**!

### Key Features:
- **Interactive Dhikr**: Beautifully designed cards for morning, evening, and sleeping prayers.
- **Virtual Garden**: Every time a child completes an Adhkar, they earn **points 💎**.
- **Shop & Plant**: Kids can spend their points to buy colorful flowers (Tulips, Roses, Sunflowers) and decorate their own garden.
- **Bonus Rewards**: Completing a whole set (like all morning prayers) grants a big **50-point bonus!**
- **Personalized Experience**: The app greets the user as a "Hero" (بطل/بطلة) and changes its theme based on the time of day.
- **Audio Support**: Listen to the correct pronunciation of each Dhikr.

## 🛠️ Tech I Used
I wanted the app to be fast and modern, so I used:
- **Next.js & React**: For a smooth, fast user interface.
- **Tailwind CSS**: For all the pretty colors and glassmorphism effects.
- **Supabase**: To keep the garden and points safe, even if the user logs out.
- **Framer Motion**: For those cute "floating" animations and transitions.

## 🚀 Getting Started
If you want to run this locally:

1. Clone the repo.
2. Run `npm install`.
3. Set up your `.env.local` with Supabase credentials.
4. Run `npm run dev`.
5. Open `localhost:3000` and start planting!
